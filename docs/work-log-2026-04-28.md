## 3. The bias finding

User pasted the **rerank prompt template** from the upstream
`pipeline/gather_data.py:418`:

> "Re-rank ... software product domains ... **Exclude non-product sites:
> review aggregators, directories, Wikipedia, news, blogs, forums, YouTube.**"

The exclusion list directly names the categories T7 (earned media) covers —
third-party press, reviews, blogs. Which means the four-method T7 finding
is partially tautological: the LLM "downranks earned media" partly because
**we told it to** in the prompt.

What survives the critique:
- T5 (keyword stuffing) — prompt doesn't mention keyword density.
- The null findings on T3/T6/T2a/T1b — no instruction in the prompt touches them.
- T7 probing (99% decodability) — that's a representational fact about
  the model, independent of prompt phrasing.

What does not survive cleanly:
- T7 DML coefficient (instruction-following confound).
- T7 ablation effect ("Official vendor page:" prefix is itself an
  authority cue, separate from earned-vs-not).

The prompt is duplicated verbatim in `interpretability/utils.py:206`
(per `docs/next-steps-prompt.md:234`: "Do not tweak the re-ranking prompt
in `utils.build_rerank_prompt`. It is transcribed verbatim from the
original `src/llm_ranker.py` — changing it invalidates the comparison
with DML.").

So the bias is **structural** — fixing it requires re-running the entire
upstream pipeline against a neutral prompt, then re-running this repo on
the new outputs.

## 7. Open questions for the paper writeup

- **Placebo arm for T7 ablation.** The "Official vendor page:" prefix is
  itself a positive cue. To separate "earned-ness was holding the page
  back" from "the prefix string just helps", run the same prefix on
  *non-earned* URLs and check whether they also rise. Cheap follow-up;
  decisive for the T7 paper claim.
- **Probing coverage.** Only T7 finished probing on the biased run. For
  the paper's Figure C to show all four treatment curves you need T5,
  T2a, T6 probing too. Re-running probing on the neutral variant gives
  the chance to fill them in. ~12 GPU-h.
- **What positive features actually work?** The current study identifies
  two demoters (T7, T5) and four nulls. To give actionable GEO advice
  ("do X to rank higher") you need a follow-up testing **promoter
  candidates** — embedding cosine similarity, content depth,
  authoritativeness markers, etc. — through the same pipeline.
